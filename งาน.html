Mind Map คลื่นแม่เหล็กไฟฟ้า

สรุปจากบทที่ 18 คลื่นแม่เหล็กไฟฟ้า (คลื่นวิทยุถึงรังสีแกมมา)
คลื่นแม่เหล็กไฟฟ้า
เกิดจากการเปลี่ยนแปลงของสนามไฟฟ้าและสนามแม่เหล็ก เคลื่อนที่ได้ในสุญญากาศด้วยความเร็ว 3 × 108 เมตร/วินาที
1. คุณสมบัติพื้นฐาน
  • เป็นคลื่นตามขวาง (สนามไฟฟ้าตั้งฉากกับสนามแม่เหล็ก)
  • ไม่ต้องอาศัยตัวกลางในการเคลื่อนที่
  • ทุกชนิดในสเปกตรัมมีความเร็วเท่ากันในสุญญากาศ (3×108 m/s)
2. สเปกตรัมของคลื่นแม่เหล็กไฟฟ้า
เรียงจากความถี่ต่ำ → สูง (ความยาวคลื่นยาว → สั้น)
  • คลื่นวิทยุ → โทรทัศน์ → ไมโครเวฟ/เรดาร์ → อินฟราเรด → แสงที่มองเห็น → อัลตราไวโอเลต → รังสีเอกซ์ → รังสีแกมมา
3. ช่วงคลื่นต่าง ๆ ในสเปกตรัม
3.1 คลื่นวิทยุ (Radio wave) 10⁶–10⁹ Hz
  • แหล่งกำเนิด: เสาอากาศวิทยุ สถานีวิทยุ
  • สมบัติ: ความยาวคลื่นยาวมาก สะท้อนชั้นไอโอโนสเฟียร์ได้ (โดยเฉพาะ AM)
  • ประโยชน์:
    • ใช้ในการกระจายเสียงระบบ A.M. และ F.M.
    • การสื่อสารวิทยุสื่อสาร
3.2 คลื่นโทรทัศน์ ~10⁸ Hz
  • ใช้ส่งสัญญาณภาพและเสียงไปยังเครื่องรับโทรทัศน์
  • ภาพส่งด้วยระบบ A.M. เสียงส่งด้วยระบบ F.M.
  • ใช้คลื่นใกล้เคียงกับคลื่นวิทยุ แต่จัดช่องความถี่เฉพาะ
3.3 คลื่นไมโครเวฟ และ เรดาร์ (Microwave / Radar) 10⁹–3×10¹¹ Hz
  • ใช้ในเตาไมโครเวฟ ทำให้น้ำในอาหารสั่น → เกิดความร้อน
  • ใช้ในระบบเรดาร์ ตรวจจับระยะและตำแหน่งของวัตถุ
  • ใช้ในการสื่อสารผ่านดาวเทียม และโทรศัพท์มือถือบางระบบ
3.4 รังสีอินฟราเรด (Infrared) 10¹¹–10¹⁴ Hz
  • เกิดจากวัตถุที่มีอุณหภูมิสูงกว่าศูนย์องศาสัมบูรณ์
  • ประโยชน์:
    • กล้องถ่ายภาพความร้อน
    • รีโมตทีวี เครื่องใช้ไฟฟ้า
    • การแพทย์ ฟื้นฟูกล้ามเนื้อด้วยความร้อน
3.5 แสงที่ตาเห็น (Visible light) ~10¹⁴ Hz
  • ช่วงความยาวคลื่นประมาณ 400–700 นาโนเมตร
  • แหล่งกำเนิด: ดวงอาทิตย์ หลอดไฟ เลเซอร์
  • ใช้ในการมองเห็น การถ่ายภาพ กล้องจุลทรรศน์
3.6 รังสีอัลตราไวโอเลต (Ultraviolet) 10¹⁵–10¹⁸ Hz
  • มาจากดวงอาทิตย์เป็นหลัก (ส่วนใหญ่ถูกชั้นโอโซนดูดกลืน)
  • ประโยชน์:
    • ฆ่าเชื้อโรค ทำให้สารบางชนิดเรืองแสง
    • ใช้ตรวจลายเซ็น ธนบัตร ปลอม
  • โทษ:
    • ทำลายเซลล์ผิวหนัง ตา → เสี่ยงมะเร็งผิวหนัง
3.7 รังสีเอกซ์ (X-ray) 10¹⁶–10²¹ Hz
  • ทะลุผ่านเนื้อเยื่ออ่อนได้ดี แต่ถูกดูดกลืนโดยกระดูกและโลหะ
  • ประโยชน์:
    • ถ่ายภาพกระดูก ตรวจหารอยร้าวโลหะ
    • ตรวจสอบสัมภาระในสนามบิน
  • โทษ:
    • ทำลายเซลล์ร่างกาย ถ้าได้รับมากเกินไปเสี่ยงมะเร็ง
3.8 รังสีแกมมา (Gamma ray) สูงกว่า X-ray
  • แหล่งกำเนิด: การสลายตัวของนิวเคลียสกัมมันตรังสี, ปฏิกิริยานิวเคลียร์
  • ประโยชน์:
    • รักษาโรคมะเร็ง (ฉายรังสี)
    • ฆ่าเชื้อในอาหาร และอุปกรณ์ทางการแพทย์
    • ปรับปรุงพันธุ์พืชในการเกษตร
  • โทษ:
    • อันตรายต่อเซลล์และสารพันธุกรรมสูงมาก อาจทำให้เกิดมะเร็งได้
4. สรุปประโยชน์และโทษของคลื่นแม่เหล็กไฟฟ้า
  • ประโยชน์ : การสื่อสาร วิทยุ โทรทัศน์ มือถือ ดาวเทียม เรดาร์ การแพทย์ ถ่ายภาพรังสี รักษามะเร็ง การเกษตร และอุตสาหกรรมต่าง ๆ
  • โทษ : คลื่นพลังงานสูงเช่น UV, X-ray, Gamma สามารถทำลายเซลล์ ทำให้เกิดแผลไหม้ มะเร็ง หรือทำลายสารพันธุกรรมได้ หากได้รับมากเกินไป
5. ทฤษฎีของแม็กซ์เวลล์ (Maxwell’s Theory)
  • สนามไฟฟ้าที่เปลี่ยนแปลงตามเวลา → ทำให้เกิดสนามแม่เหล็ก
  • สนามแม่เหล็กที่เปลี่ยนแปลง → ทำให้เกิดสนามไฟฟ้า
  • การเหนี่ยวนำซึ่งกันและกันของสองสนามนี้ทำให้เกิด คลื่นแม่เหล็กไฟฟ้า
  • คลื่นสามารถเคลื่อนที่ได้ในสุญญากาศ โดยไม่ต้องมีตัวกลาง
  • ความเร็วของคลื่นแม่เหล็กไฟฟ้า (ในสุญญากาศ)
  • c = 3 × 108 m/s
  • สูตรความสัมพันธ์: c = fλ
  • ทิศทางของสนาม
  • สนามไฟฟ้า (E) ⟂ สนามแม่เหล็ก (B)
  • และทั้งสองสนาม ⟂ ทิศทางการเคลื่อนที่ของคลื่น
6. สมการคลื่นแม่เหล็กไฟฟ้า (ออกสอบบ่อย)
  • 1) สูตรหาความเร็วคลื่น (v)
  • v = fλ
  • ความหมาย:
    • v = ความเร็วคลื่น (m/s)
    • f = ความถี่ (Hz)
    • λ = ความยาวคลื่น (m)
  • 2) สูตรหาความถี่จากคาบ
  • f = 1 / T
  • ความหมาย:
    • f = ความถี่ (Hz)
    • T = คาบ (s) → เวลาที่คลื่นสั่นครบ 1 รอบ
  • 3) สูตรหาคาบจากความถี่
  • T = 1 / f
  • 4) ความเร็วคลื่นแม่เหล็กไฟฟ้าในสุญญากาศ
  • c = 3 × 108 m/s
  • 5) สรุปสูตรจำไปสอบ
  • v = fλ
  • f = 1/T
  • T = 1/f
  • c = 3×108 m/s
  • ตัวอย่างโจทย์ที่ออกสอบบ่อย
  • โจทย์ให้ f และ λ → หาค่า v
  • โจทย์ให้ T → หาค่า f
  • โจทย์ให้ f → หาค่า T
